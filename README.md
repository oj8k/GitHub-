4. 通过审核后合并到主分支

**格式要求**：
```yaml
- category: "01、AI应用生态"
- subcategory: "AI工具"
- name: "LanguageTool"
- description: "多语言语法校验工具"
- repo: "https://github.com/languagetool-org/languagetool"
```
